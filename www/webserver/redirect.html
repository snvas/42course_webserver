Redirect

Um Redirect, ou redirecionamento, é uma maneira de enviar os visitantes de uma URL para outra automaticamente. Isso pode ser útil em várias situações, como quando um site é movido para um novo domínio ou uma página específica foi alterada para um novo endereço.

Há diferentes tipos de redirecionamentos, como o redirecionamento 301 (permanente) e o redirecionamento 302 (temporário).
