Métodos GET, POST e DELETE

Esses métodos são comuns em HTTP e definem as operações que podem ser realizadas em um recurso:

  • GET: Solicita dados de um recurso. É usado para consultar informações sem causar efeitos colaterais.
  • POST: Envia dados para serem processados por um recurso. É comumente usado para enviar informações de um formulário para ser processado por um servidor.
  • DELETE: Solicita a remoção de um recurso.
