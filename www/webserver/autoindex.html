O que é o Autoindex?

O Autoindex é uma funcionalidade que muitos servidores web possuem que, quando habilitada, lista os conteúdos de um diretório quando não há um arquivo index (como index.html) presente. Isso pode ser útil para permitir a navegação de arquivos, mas também pode representar um risco de segurança se não for usado intencionalmente.
